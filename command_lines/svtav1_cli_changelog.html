from 10/28/2019 --------------------

Pass 1:

./SvtAv1EncApp -i <input YUV> -o <reconstructed YUV> -w <width> -h <height> -q <qp> -fps-num <fps num> -fps-denom <fps denom> -n <number of frames> -b <output bitstream> -intra-period <kf interval> -bit-depth <bit depth> -c config/Sample.cfg -scd 0 -enc-mode 5 -output-stat-file fp_stats.stat -enc-mode-2p 0 -lp 48

Pass 2:

./SvtAv1EncApp -i <input YUV> -o <reconstructed YUV> -w <width> -h <height> -q <qp> -fps-num <fps num> -fps-denom <fps denom> -n <number of frames> -b <output bitstream> -intra-period <kf interval> -bit-depth <bit depth> -c config/Sample.cfg -scd 0 -enc-mode 0 -input-stat-file fp_stats.stat -lp 48

<kf interval> = -1 (i.e. only first key-frame)

<qp> = {20, 32, 43, 55, 59}
